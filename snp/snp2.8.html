Sutta Nipata Chapter 2: The Minor Chapter - Culavagga

  • 2. The Minor Chapter
  • 3. The Great Chapter
  • 4. The Eights
  • 5. Way to the Beyond

318. because (or from which) bhi the dhamma man vija other (or final knowledge)
indaṃ'and to/for that (or to/for him) devas might (or may/would or to be done) causes to pūjati,
so pūjito in (or on/at/within/herein/among) that (or him) pasannacitto
well-learned (much learning) pātukaroti the dhamma.

319. that having aṭṭhikati nisamma the wise
dammānudhammaṃ (“|”) (ing) is being (or ing if present verb usage) paṭipajjati,
wise vibhāvī nipuṇo and comes to be (or becomes or is)
which (or he/she who) tādisaṃ bhajate vigilant.

320. and khuddaṃ bālaṃ (“|”) (ing) is being (or ing if present verb usage) upasevati
and anāgatatthaṃ and usūyakaṃ,
only (or also or even or simply or just) here (this world) the dhamma having causes to acibhāvati
avitiṇṇakaṅkho death arrives at (or enter into or engage). bjt page 96

321. as (or how, in which way) naro āpagaṃ having otarati
mahodakaṃ salilaṃ sīghasotaṃ,
so (“|”) (ing) is being (or ing if present verb usage) vuyhati I (must) anusotagati
what is so pare pakkati to tārayeti.

322. tathecha the dhamma having causes to avibhāvati
of/for bahussuta (s) you (aorist of) causes to anisāmati,
oneself (or self, or in the evening) of/for aja (s) avītiṇṇakaṅkho
what is so pare sakkati to convinces (or wins over).

323. as āpi nāvaṃ having daḷhamāruhati
by which (or with) dear (affection/beloved)'by which (or with) ritta provided with,
so tāraye so (there or in that place) bahūpi other (or another)
tatrūpāyaññū wholesome (or skilled) mutīmā. pts page 056

324. thus too (even/just so) which (or he/she who) a knowledge master bhāvitatto
well-learned (much learning) comes to be (or becomes or is) avedhadhammo,
so kho pare nijjhapaye of/for paja (s)
sotāvadhānūpanīsūpanne.

325. therefore (or from that) a being true man you (all) (must) bhajeti
only (or also or even or simply or just) and medhāvinaṃ and bahussutaṃ,
having known benefit (or meaning) (“|”) (ing) is being (or ing if present verb usage) paṭipajjati
viññātadhammo so pleasant you (all) (must) labheti (“|”).

318. yasmā bhi dhammaṃ puriso vijaññā
indaṃ'ca naṃ devatā pūjayeyya,
so pūjito tasmiṃ pasannacitto
bahussuto pātukaroti dhammaṃ.

319. tadaṭṭhikatvāna nisamma dhīro
dammānudhammaṃ paṭipajjamāno,
viññū vibhāvī nipuṇo ca hoti
yo tādisaṃ bhajate appamatto.

320. khuddañca bālaṃ upasevamāno
anāgatatthañca usūyakañca,
idheva dhammaṃ acibhāvayitvā
avitiṇṇakaṅkho maraṇaṃ upeti. bjt page 96

321. yathā naro āpagaṃ otaritvā
mahodakaṃ salilaṃ sīghasotaṃ,
so vuyhamāno anusotagāmi
kiṃ so pare pakkati tārayetuṃ.

322. tathecha dhammaṃ avibhāvayitvā
bahussutānaṃ anisāmayattha,
sayaṃ ajānaṃ avītiṇṇakaṅkho
kiṃ so pare sakkati nijjhāpetuṃ.

323. yathāpi nāvaṃ daḷhamāruhitvā
piyena'rittena samaṅgibhūto,
so tāraye tattha bahūpi aññe
tatrūpāyaññū kusalo mutīmā. pts page 056

324. evampi yo vedagū bhāvitatto
bahussuto hoti avedhadhammo,
so kho pare nijjhapaye pajānaṃ
sotāvadhānūpanīsūpanne.

325. tasmā bhave sappurisaṃ bhajetha
medhāvinañceva bahussutañca,
aññāya atthaṃ paṭipajjamāno
viññātadhammo so sukhaṃ labhethāti.
